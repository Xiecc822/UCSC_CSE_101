# 2023 Fall CSE 101 PA6     


## Description:          
    
    In this assignment, I created a BigInteger ADT in C++ that is capable 
    of performing arithmetic operations on arbitrarily large signed integers.


## Files:    
* Arithmetic.cpp    
- BigInteger.cpp 
* BigInteger.h    
- BigIntegerTest.cpp
* List.cpp 
- List.h 
* ListTest.cpp    
- Makefile 
* README.md     
   
   
## Build    
    
    $ make     
    
    or
    
    $ make ListTest     
    
    or
    
    $ make BigIntegerTest
    
   
## Running   
            
    $ ./Arithmetic <input file> <output file>        
    
    $ ./ListTest           
    
    $ ./BigIntegerTest        
       

## Cleaning    
    
    $ make clean
