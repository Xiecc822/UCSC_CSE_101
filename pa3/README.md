# 2023 Fall CSE 101 PA3     


## Description:          
    
    The goal for this project is to build a Graph module
    in C that implements the Depth First Search (DFS) algorithm.


## Files:    
* List.h    
- List.c 
* Graph.h
* Graph.c    
- FindComponents.c
* GraphTest.c 
- Makefile 
* README.md
   
   
## Build    
    
    $ make     
    
    or
    
    $ make GraphTest
    
   
## Running   
            
    $ ./FindComponents <input file> <output file>    
    
    $ ./GraphTest          
       

## Cleaning    
    
    $ make clean
