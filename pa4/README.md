# 2023 Fall CSE 101 PA4     


## Description:          
    
    The goal for this project is to create a calculator for 
    performing matrix operations that exploits the (expected) 
    sparseness of its matrix operands.


## Files:    
* Sparse.c    
- Matrix.c 
* Matrix.h 
- MatrixTest.c 
* List.c
- List.h 
* ListTest.c
- Makefile 
* README.md
   
   
## Build    
    
    $ make     
    
    or
    
    $ make ListTest
    
    or
    
    $ make MatrixTest
    
   
## Running   
            
    $ ./Sparse <input file> <output file>    
    
    $ ./ListTest    
    
    $ ./MatrixTest               
       

## Cleaning    
    
    $ make clean
