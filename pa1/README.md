# 2023 Fall CSE 101 PA1     


## Description:          
    
    The goal for this project is to build an Integer List ADT
    in C and use it to indirectly alphabetize the lines in a file.


## Files:    
* List.h    
- List.c 
* ListTest.c
* Lex.c    
- Makefile 
* README.md
   
   
## Build    
    
    $ make     
    
    or
    
    $ make ListTest
    
   
## Running   
            
    $ ./Lex <input file> <output file>    
    
    $ ./ListTest          
       

## Cleaning    
    
    $ make clean
