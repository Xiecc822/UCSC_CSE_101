# 2023 Fall CSE 101 PA8     


## Description:          
    
    In this project, I re-created the Dictionary ADT from pa7, 
    but now based on a Red-Black Tree.

## Files:    
* README.md    
- Makefile 
* Dictionary.h    
- Dictionary.cpp
* DictionaryTest.cpp 
- Order.cpp 
* WordFrequency.cpp   
   
   
## Build    
    
    $ make     
    
    or
    
    $ make WordFrequency     
    
    or
    
    $ make Order     
    
    or
    
    $ make DictionaryTest     
    
   
## Running   
            
    $ ./WordFrequency <input file> <output file>        
    
    $ ./Order <input file> <output file>        
    
    $ ./DictionaryTest                        
       

## Cleaning    
    
    $ make clean
